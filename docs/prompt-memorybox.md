# MemoryBox - 本地精美电子相册

## 项目提示词

使用 `/antigravity-team` 工作流执行以下需求：

---

```
/antigravity-team

【项目名称】MemoryBox - 本地精美电子相册

【需求描述】
创建一个功能丰富的本地电子相册 Web 应用，用于管理和展示珍贵回忆。

【核心功能】

1. 照片管理
   - 批量上传照片（支持拖拽）
   - 照片编辑（裁剪、滤镜、旋转）
   - 相册分类和标签管理
   - 照片排序和搜索

2. 短视频支持
   - 上传和播放短视频（MP4/WebM）
   - 视频封面自动提取
   - 视频与照片混合展示

3. AI 智能文案
   - 调用大模型为照片生成描述文字
   - 支持多种文案风格（浪漫/文艺/幽默/简约）
   - 一键生成整个相册的故事文案

4. 多种现代化样式
   - 至少 5 种相册主题模板（极简白/暗黑/复古胶片/杂志风/ins风）
   - 整体页面风格一键切换
   - 动态字体效果（打字机/渐入/弹跳）
   - 照片展示动画（淡入/滑动/缩放/翻转）

5. 纪念日功能
   - 添加重要纪念日和日期
   - 每个纪念日的倒计时/正计时显示
   - 纪念日到达时的特效提醒
   - 与相关照片关联

6. 页面布局
   - 瀑布流/网格/时间轴多种布局
   - 响应式设计，支持移动端
   - 全屏幻灯片播放模式

【技术要求】

- 前端：React/Vue + TypeScript
- 后端：Node.js/Python (FastAPI)
- 存储：本地文件系统 + SQLite
- AI：调用 OpenAI/Claude API 生成文案
- 部署：Docker + docker-compose 一键部署

【非功能性需求】

- 纯本地运行，数据不上传云端
- 首页加载时间 < 2s
- 支持 1000+ 张照片的流畅浏览
- 移动端友好

【验收标准】

- [ ] 可成功上传和展示照片/视频
- [ ] AI 文案生成功能正常工作
- [ ] 至少 3 种主题可切换
- [ ] 纪念日倒计时正确显示
- [ ] Docker 部署成功，浏览器可访问

【排除范围】

- 云端同步功能
- 用户登录系统
- 照片打印服务
```
