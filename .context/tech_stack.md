# 技术栈说明

## 后端
- **Python 3.10+** - 主要开发语言
- **FastAPI** - API 框架（推荐）
- **Pydantic** - 数据验证

## 前端
- **TypeScript** - 类型安全
- **React/Vue 3** - UI 框架
- **Vite** - 构建工具

## 基础设施
- **Docker** - 容器化
- **GitHub Actions** - CI/CD
- **PostgreSQL** - 数据库（推荐）

## AI/MCP
- **MCP Protocol** - 工具集成
- **DeepSeek** - LLM 支持
