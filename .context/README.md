# 动态上下文目录

此目录下的所有 `.md` 文件会被 AI 在每次任务开始时自动读取。

## 使用方法

1. 创建 `xxx.md` 文件
2. 写入需要 AI 持续了解的信息
3. AI 会在工作流启动时自动注入这些知识

## 示例用途

- `tech_stack.md` - 项目技术栈说明
- `api_conventions.md` - API 命名规范
- `deployment.md` - 部署相关信息
- `team_conventions.md` - 团队约定

## 注意事项

- 文件内容会增加上下文长度，保持精简
- 仅放置稳定的、需要长期遵循的信息
- 临时信息请使用 `mission.md` 或任务目录
